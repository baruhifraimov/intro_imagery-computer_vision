# Introduction to Imagery - Ex1

This exercise implements basic image processing tasks in Python using NumPy, OpenCV, and Matplotlib.

## What it does

- Reads and displays images in grayscale or RGB.
- Converts images between RGB and YIQ color spaces.
- Performs histogram equalization.
- Performs optimal image quantization and shows the error per iteration.
- Includes a simple gamma-correction viewer with a slider.
